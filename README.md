# 🚀 SmileTech Training & Project Management System

A comprehensive **Java Full Stack web application** developed to manage training programs, internships, workshops, webinars, student projects, and company services. The system provides separate **Admin** and **User** dashboards with secure role-based authentication, enabling efficient management of students, courses, applications, feedback, and company information.

---

## 📌 Project Overview

SmileTech Training & Project Management System is designed for a training and project development company that offers professional services to engineering and college students.

Students can:
- Register and Login
- Browse available courses
- Apply for internships
- Apply for project domains
- Submit project applications
- View company information
- Browse the gallery
- Submit feedback
- Contact the company

Administrators can:
- Manage students
- Manage courses
- Manage internships
- Manage project domains
- Manage workshops
- Manage webinars
- Manage training programs
- Manage company gallery
- Manage feedback
- Manage company information
- Monitor dashboard statistics
